Proyectos

Proyecto 1

En mi primer proyecto tuve la oportunidad de hacerlo junto a mi compañero el SR. Jeremy Dominguez, hicimos un sistema de facturación usando html para las etiquetas, css para los estilos, JavaScript para la interacción, MySQL Workbench para la base de datos más Nove.js, la función del proyecto es báasicamente facilitar las funciones básicas de una empresa, los módulos interactivos que tiene el sistema son: Ventas, Cobros, Inventario, Contabilidad. En el módulo de Ventas aparecen todos los cobros del día que se hallan echo, en el módulo de cobros se hacen todos los cobros a clientes, el módulo de inventarion esta todo lo relacionado con la mercancía dandole una mejor manejo de todos los productos y por último el módulo de Contabilidad donde aqui le damos un mejor manejo y claridad de todas las finanzas de la empresa.
